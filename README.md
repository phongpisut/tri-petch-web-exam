# tri-petch-web-exam

## Libraries

| Name                      |           Usage            |
| :------------------------ | :------------------------: |
| styled-components         | organize styled components |
| tailwindcss               |   CSS and media queries    |
| react-scroll-parallax     | parallax effect for images |
| react-responsive-carousel |    responsive carousel     |

## Start Project

```
 cd tri-petch-web-exam
 --
 yarn
 or
 npm install
 or
 pnpm install
 --
 yarn dev
 or
 npm run dev
 or
 pnmp run dev
 --

---
http://localhost:5173/
```
